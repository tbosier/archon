# Security Policy

## Reporting a vulnerability

Please do not report security vulnerabilities in a public issue.

Use GitHub's private vulnerability reporting form:

https://github.com/tbosier/archon/security/advisories/new

Include the affected version, reproduction steps, likely impact, and any
suggested fix. Remove credentials and personal data from logs before attaching
them.

You should receive an acknowledgement within seven days. A fix timeline depends
on severity and whether the issue affects an upstream provider CLI.

## Supported versions

Archon is currently pre-release software. Security fixes are applied to the
latest commit on `main` until versioned releases begin.
